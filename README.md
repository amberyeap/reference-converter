# Academic Reference Reformatter
This project takes APA-style references and reformats it to meet Malaysia's National University (UKM)'s citation format.

## [Demo](https://reference-convert.vercel.app/)
<img width="1918" height="909" alt="image" src="https://github.com/user-attachments/assets/6b56a420-da0e-4118-94c3-74fe74674679" />


## Current Features
Since the project is still in the very early development stage, the features are very limited.

- Converts a single APA reference into UKM format
- Sorts the converted references alphabetically based on authors' names, then by year

## Planned Features
- Multiple references
- Edit and manage a saved reference list
- Copy-to-clipboard button
- and more...

## Usage
Example Input
<pre>Smith J., Doe A. (2023). Interesting Study. Journal of Testing, 12(5), 235–543.</pre>
Example Output
<pre>Smith J., Doe A. 2023. Interesting Study. Journal of Testing 12(5): 235–543.</pre>

This project was built to ease my sister's frustrations of having to manually modify her reference lists.
